# Zeyu Homepage

This is a lightweight static homepage built from the information in
`D:\BaiduSyncdisk\学习\申博`.

## Files

- `index.html`: main page
- `assets/styles.css`: styling
- `assets/site.js`: small animation and last-updated helper
- `assets/media/`: local avatar and demo clip
- `assets/papers/`: CV and selected paper PDFs
- `assets/previews/`: generated paper cover images

## Open locally

Double-click `index.html` or open it in any browser.
